# 📄 Scripts

Clone the repository:

```bash
$ git clone https://github.com/jonathangaldino/portfolio.git
```

Install dependencies, use inside the folder:

```bash
$ yarn
```

## 🔨 Development


Start the development server

```bash
$ yarn dev
```
